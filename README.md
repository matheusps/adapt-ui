### 👨🏾‍💻 Develop

```bash
# install dependencies
$ yarn

# start development server with storybook
$ yarn storybook

# Build the docs
$ yarn build-storybook

# Build the lib
$ yarn build-lib

# Build lib and docs
$ yarn build

# Run tests
$ yarn test

# Create a new release with np
$ yarn release
```
